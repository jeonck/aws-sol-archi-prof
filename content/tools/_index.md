이 섹션은 [학습 로드맵](../docs/)에서 다루는 이론과 실습을 직접 손으로 확인할 때 사용할 수 있는 **학습 플랫폼**, **강의·모의고사**, **공식 표준·참고 문서**를 한 곳에 모아둔 참고 자료실입니다.

`/docs`의 각 페이지는 "왜, 무엇을" 설명하는 데 집중하고, 이 섹션은 "어떤 도구로, 어디서 더 읽을 수 있는가"에 집중합니다.

## 사용 방법

1. [학습 로드맵](../docs/)을 따라가며 개념을 먼저 익힙니다.
2. 해당 주제와 관련된 도구가 등장하면 이 섹션에서 설치 방법과 기본 사용법을 확인합니다.
3. [실습 (Labs)](../labs/) 섹션에서 실제 콘솔과 코드로 직접 실행해봅니다.

{{< callout type="warning" >}}
이 섹션에 소개된 도구와 자료는 모두 외부 플랫폼 또는 공식 기관의 자료입니다. URL, 가격, 라이선스는 시간이 지나면 바뀔 수 있으니 링크를 통해 항상 최신 정보를 직접 확인하세요.
{{< /callout >}}
