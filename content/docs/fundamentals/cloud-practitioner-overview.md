AWS Certified Cloud Practitioner(CLF-C02)는 SAA-C03이나 SAP-C02를 목표로 한다면 필수 코스는 아닙니다. 하지만 클라우드 경험이 거의 없는 입문자에게는 강력하게 추천되는 출발점입니다. 시험 자체보다 중요한 것은 이 과정에서 따라오는 **무료 학습 콘텐츠와 클라우드 공통 어휘**입니다.

## 왜 선택이지만 추천인가

CLF-C02는 기술 난이도가 낮고 비기술 영역(과금 모델, 지원 플랜, 거버넌스)을 포함하기 때문에 SAA 시험 범위와 완전히 겹치지는 않습니다. 그래서 "SAA를 빨리 따고 싶다"는 목표만 있다면 건너뛸 수 있습니다.

그럼에도 추천하는 이유는 세 가지입니다.

1. **어휘 습득** — 리전, 가용 영역, 온디맨드/예약/스팟, 책임 공유 모델(Shared Responsibility Model) 같은 용어를 SAA 학습 전에 익혀두면 본 학습 속도가 훨씬 빨라집니다.
2. **전체 그림** — AWS의 서비스 카탈로그가 워낙 방대하기 때문에, 초반에 "이 회사가 무엇을 파는지" 지도를 한 번 그려보는 과정이 이후 학습의 맥락을 잡아줍니다.
3. **심리적 진입 장벽 완화** — 시험 난이도가 낮아 첫 합격 경험을 빠르게 만들 수 있고, 이는 이후 SAA·SAP 같은 장기 학습을 지속하는 동기가 됩니다.

{{< callout type="info" >}}
**비전공자, 첫 IT 자격증인 경우** CLF-C02부터 시작하는 것을 권장합니다. 반대로 개발/인프라 경력이 있다면 [SAA-C03](../../saa/)부터 바로 시작해도 무리가 없습니다.
{{< /callout >}}

## AWS Skill Builder 무료 코스

[AWS Skill Builder](https://skillbuilder.aws/)는 AWS가 직접 운영하는 학습 플랫폼으로, CLF-C02 대비 과정을 포함한 다수의 무료 디지털 코스를 제공합니다.

- **AWS Cloud Practitioner Essentials** — 약 6시간 분량의 공식 입문 코스로, 컴퓨팅·스토리지·네트워킹·보안 기본 개념을 서비스 단위로 설명합니다.
- **Exam Prep 시리즈** — 시험 도메인별 표준 문항과 공식 학습 가이드를 제공하며, 실제 시험과 유사한 문제 유형을 미리 경험할 수 있습니다.
- 계정만 만들면 비용 없이 모든 기초 코스에 접근할 수 있어, 자격증 응시 여부와 무관하게 첫 학습 자료로 적합합니다.

## AWS Cloud Quest — 게임형 학습

[AWS Cloud Quest](https://aws.amazon.com/training/digital/aws-cloud-quest/)는 3D 게임 형태로 실제 AWS 콘솔 작업을 수행하며 배우는 학습 도구입니다. 캐릭터를 키우고 퀘스트를 깨는 형식이라 텍스트 강의보다 몰입도가 높고, 무엇보다 **실제 콘솔에서 리소스를 생성·삭제하는 손동작**을 게임 안에서 안전하게 반복할 수 있습니다.

Cloud Practitioner 트랙은 무료로 제공되며, Solutions Architect·Developer 트랙은 일부 구독 모델로 운영됩니다. 입문 단계에서는 Cloud Practitioner 트랙만으로도 충분히 가치가 있습니다.

## 학습 시간 추정

| 배경 | 권장 학습 기간 | 주당 투입 시간 |
|---|---|---|
| IT/개발 경력 보유 | 1~2주 | 3~5시간 |
| 비IT 직군, 입문자 | 3~4주 | 5~8시간 |

CLF-C02는 객관식 65문항, 90분 시험으로 난이도 자체는 높지 않습니다. 다만 이 단계에서 시간을 아끼겠다고 핵심 서비스 개념을 대충 넘기면, 다음 단계인 [AWS 핵심 서비스](../aws-core-services/)에서 다시 같은 내용을 복습해야 하므로 오히려 비효율적입니다.

{{< callout type="warning" >}}
CLF-C02 합격증 자체보다 중요한 것은 Skill Builder와 Cloud Quest를 통해 **콘솔 화면을 직접 눌러본 경험**입니다. 시험만 보고 콘솔을 한 번도 안 켜봤다면, 다음 단계인 [Free Tier 계정 생성](../free-tier-account-setup/)에서 반드시 직접 실습으로 보완하세요.
{{< /callout >}}
