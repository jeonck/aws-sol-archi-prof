자료가 너무 많으면 오히려 학습 속도가 떨어집니다. 이 페이지는 검증된 자료를 역할별로 좁혀서 추천하고, 각 자료를 학습 흐름의 어느 시점에 사용해야 하는지 안내합니다.

## AWS Skill Builder — 4단계 Exam Prep Plan

[AWS Skill Builder](https://skillbuilder.aws/)는 AWS가 공식 운영하는 학습 플랫폼으로, SAA-C03와 SAP-C02 모두 전용 **Exam Prep Plan**을 제공합니다. 이 플랜은 4단계로 구성되어 있고, 순서대로 진행하는 것이 설계 의도입니다.

1. **사전 진단 문제(Official Pretest)** — 학습을 시작하기 전 현재 수준을 가늠하는 진단 문제를 풉니다. 어떤 도메인이 약한지 미리 파악할 수 있습니다.
2. **공식 코스(Exam Prep Standard Course)** — 도메인별로 구성된 공식 학습 코스를 진행합니다. AWS 공식 자료이기 때문에 시험 출제 범위와 용어가 가장 정확히 일치합니다.
3. **도메인별 복습 문제(Official Practice Question Sets)** — 각 도메인 학습 후 해당 도메인만 다루는 연습 문제로 즉시 복습합니다.
4. **준비도 확인(Official Practice Exam)** — 전체 학습을 마친 뒤 실제 시험과 동일한 형식의 풀타임 모의고사로 최종 준비도를 확인합니다.

{{< callout type="info" >}}
Skill Builder의 일부 콘텐츠는 무료지만, Exam Prep Plan의 Practice Exam 일부와 심화 코스는 유료 구독(Individual 또는 Team)이 필요합니다. 예산이 제한적이라면 무료 진단 문제와 공식 코스만으로도 학습 방향을 잡는 데 충분합니다.
{{< /callout >}}

## Udemy 강의 — Stephane Maarek / Adrian Cantrill

영어 강의에 거부감이 없다면, 다음 두 강사의 강의가 SAA/SAP 학습자 사이에서 가장 널리 추천됩니다.

- **Stephane Maarek** — SAA-C03, SAP-C02 강의 모두 진행하며, 핵심 개념을 빠르게 짚고 실습(Hands-on) 데모를 충실히 포함합니다. 분량이 상대적으로 짧아 시간이 제한적인 학습자에게 적합하고, 강의에 포함된 Practice Test도 평가가 좋습니다.
- **Adrian Cantrill** — 강의가 훨씬 깊고 길지만(SAA 기준 80시간 이상), 단순 시험 대비를 넘어 실무에서 바로 쓸 수 있는 깊이의 설명을 제공합니다. 시간 여유가 있고 "시험만 통과하는 것"이 아니라 실력을 제대로 쌓고 싶은 학습자에게 추천됩니다.

두 강의 모두 Udemy 할인 기간(거의 상시 진행)에 구매하면 합리적인 가격에 접근할 수 있습니다.

## Tutorials Dojo Practice Exams

[Tutorials Dojo](https://tutorialsdojo.com/)는 AWS 자격증 Practice Exam 분야에서 가장 신뢰받는 제3자 제공처입니다.

- 문제 난이도가 실제 시험과 가깝게 보정되어 있다는 평가가 많고, 오답마다 상세한 해설(왜 정답이고 왜 나머지가 틀렸는지)을 제공합니다.
- SAA-C03, SAP-C02 모두 별도 문제 세트가 있으며, 도메인별 점수 분석 리포트를 제공해 약점 파악에 유용합니다.
- **[Practice Exam 전략](../practice-exam-strategy/)** 에서 설명한 매주 루틴에 가장 적합한 자료 중 하나입니다.

## Well-Architected Framework Whitepaper

[AWS Well-Architected Framework 공식 백서](https://docs.aws.amazon.com/wellarchitected/latest/framework/welcome.html)는 무료로 제공되며, SAA와 SAP 모두에서 직접적으로 출제 근거가 되는 핵심 자료입니다. 6 Pillars(운영 우수성, 보안, 안정성, 성능 효율성, 비용 최적화, 지속가능성) 각각의 설계 원칙을 공식 용어로 정리하고 있어, 시험에서 등장하는 "trade-off 판단" 문제의 사고 틀을 그대로 제공합니다. 시험 직전 1~2회독을 권장합니다.

## re:Invent 세션

[AWS re:Invent](https://reinvent.awsevents.com/) 세션 영상은 YouTube에 무료로 공개되며, 특히 다음 두 트랙이 시험 학습에 직접적으로 도움이 됩니다.

- **Architecture 트랙** — 실제 고객 사례를 기반으로 한 아키텍처 설계 세션은 SAP-C02의 시나리오형 문제와 사고 구조가 매우 유사합니다.
- **Well-Architected 관련 세션** — 매년 업데이트되는 Pillar별 심화 세션은 백서보다 더 구체적인 실전 사례를 다룹니다.

시간이 부족하다면 강의나 백서를 우선하고, re:Invent 세션은 특정 주제(예: 멀티 리전 DR)를 더 깊이 이해하고 싶을 때 보조 자료로 활용하세요.
